⚠ Upstash Redis not configured — tracking is inactive

To enable live tracking, create a free Upstash Redis database at upstash.com, then add these two environment variables to your Vercel project:

UPSTASH_REDIS_REST_URL — the REST URL from your Upstash database page

UPSTASH_REDIS_REST_TOKEN — the Read/Write token from your Upstash database page

After adding the env vars, redeploy and data will start flowing immediately.
